
regular controls:
wasd to move.
ENTER to start game or to return to start screen after death.



Debug keys:

H - Spawn ghosts.

E - Enrage red ghost (two levels).

M - New level.

L - Give player extra life.

X - Kill player.

T - Toggle scatter and hunt modes for ghosts.

F - Energize player.

N - Toggle node map view.

R - Spawn Fruit.
